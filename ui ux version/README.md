# Loteo Pargua Sur · React + Vite

Catálogo web React para venta de loteo industrial entre Puerto Montt y Pargua, Región de Los Lagos, Chile.

Construido con el skill `ui-ux-pro-max`. Estilo: Swiss Modernism + Neubrutalist. Paleta Construction/Architecture (gris industrial + naranja seguridad). Tipografía Space Grotesk + DM Sans.

## Setup

```bash
npm install
npm run dev
```

Abre `http://localhost:5174`.

## Build

```bash
npm run build
npm run preview
```
